== Description ==
SF Contact Form is a contact form that integrates with the SalesForce Web-2-Lead function. It can be
implemented on a page or a post. 
This plugin is based heavily on the original [WP Contact Form][wpcontact syntax].

[wpcontact syntax]: http://blog.ftwr.co.uk/wordpress/
            "WP Contact Form"

For further details on the plugin and to post a comment with any change requests, please visit my [plugin page here][cuplaweb page]
[cuplaweb page]: http://www.cuplaweb.com/software/wordpress-plugins/sf-contact-form/
            "SF Contact Homepage"

== Installation ==

1. Upload to your plugins folder, usually `wp-content/plugins/`
2. Activate the plugin on the plugin screen.
3. Configure the plugin on it's settings screen. Settings ... SF Contact Form

== Frequently Asked Questions ==

= How do I add the contact form to a post/page? =

You need to add the `<!--sfcontact form-->` to the body of the post/page in the editors HTML mode.
